# 🇸🇦 Saudi Food Delivery Market Dashboard

https://food-delivery-dashboard-kfc7csdfqm5cdtyofudmzu.streamlit.app/

📊 **Interactive Dashboard for Saudi Food Delivery Market Analysis**  

This is a **Streamlit dashboard** that provides insights into major food delivery platforms in Saudi Arabia, including:

- HungerStation  
- Jahez  
- Keeta  
- Ninja  

> This version **does not include order counts** and focuses on:
> **Revenue, Active Users, Average Order Value, Customer Retention, Market Share, Marketing Spend.**

---

## 🔹 Features

- **Interactive KPI Cards**:
  - Total Revenue  
  - Average Order Value  
  - Customer Retention Rate  
  - Total Active Users  
  - Marketing Spend  

- **Charts and Visualizations**:
  - Revenue by Platform (Bar Chart)  
  - Average Order Value by Platform (Bar Chart)  
  - Active Users by Platform (Bar Chart)  
  - Market Share by Platform (Pie Chart)  
  - Marketing Spend by Platform (Bar Chart)  

- **Platform Filter**: Filter data by HungerStation, Jahez, Keeta, Ninja, or all platforms.  

- **Interactive Data Table**: View the dataset used for the analysis.  

- **Professional Theme**: Blue & white clean business-style dashboard.
